# Contributing

## Setup

```bash
chmod +x setup.sh download-model.sh run-whisper-dictate-pl.command scripts/build_macos_dist.sh
./setup.sh
```

## Development Workflow

1. Make focused changes.
2. Keep repository artifacts out of git.
3. Verify Python files compile.
4. If you change packaging, rebuild the macOS app.

## Release Workflow

Use the documented process in `docs/RELEASE.md`.
